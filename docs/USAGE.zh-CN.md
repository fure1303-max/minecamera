# MineCamera 使用说明

## 版本要求

- Minecraft：`1.21.11`
- Fabric Loader：`0.19.3`
- Java：`21`

`MineCamera` 当前按以上版本组合进行开发和验证，不按“最新版 Fabric 通用兼容”承诺发布。

## 物品说明

- `相机`：打开 HUD 取景、拍照和回放照片
- `三脚架`：放置在世界中后可挂载相机，作为独立机位使用
- `胶卷`：空白胶卷用于拍摄，已曝光胶卷用于回放和展示
- `照片展示框`：可放置单张照片，也可自动拼成大图展示组

## 基础操作

### 手持相机

- `右键`：打开相机 HUD

### 相机取景模式

- `左键`：拍照
- `右键`：关闭相机 HUD
- `Tab`：切换分辨率预设
- `鼠标滚轮`：调整焦距
- `Shift + 鼠标滚轮`：快速调整焦距
- `R`：切换到回放模式

### 相机回放模式

- `A`：上一张照片
- `D`：下一张照片
- `R`：返回取景模式
- `右键`：关闭相机 HUD

## 三脚架操作

- 手持 `三脚架` `右键`：放置三脚架
- 手持 `相机` 对空三脚架 `Shift + 右键`：安装相机
- 对已安装相机的三脚架 `右键`：打开三脚架相机
- 空手对已安装相机的三脚架 `Shift + 右键`：拆下相机

## 胶卷规则

- 背包里有 `空白胶卷` 时才能拍照
- 每次成功拍照会消耗 `1` 个空白胶卷
- 拍照后会生成 `1` 个已曝光胶卷
- 已曝光胶卷可在相机 `回放模式` 中查看

## 照片展示框

- 手持 `照片展示框` `右键`：放置展示框
- 手持 `已曝光胶卷` 对展示框 `右键`：展示照片
- 多个 `朝向一致` 且 `连续相邻` 的展示框会自动组成大图展示组

## 合成配方

### 相机

```text
玻璃板  铁锭    玻璃板
铁锭    钻石    铁锭
空      铁锭    空
```

### 三脚架

```text
空      铁锭    空
空      木棍    空
木棍    空      木棍
```

### 胶卷

无序合成：

```text
纸 + 纸 + 黑色染料 = 胶卷 x2
```

### 照片展示框

```text
木棍    纸      木棍
纸      空      纸
木棍    纸      木棍
```

## 照片保存位置

通用路径：

```text
<游戏目录>\versions\<当前版本>\minecamera\captures\
```

照片文件格式：

```text
<mediaId>.png
```

## 常见问题与日志路径

游戏日志：

```text
<游戏目录>\versions\<当前版本>\logs\latest.log
```

崩溃报告：

```text
<游戏目录>\versions\<当前版本>\crash-reports\
```

如果启动器把报错导出到临时目录，也可能在：

```text
C:\Users\<用户名>\AppData\Local\Temp\
```

## 测试建议

- 拍照前先确认背包里有空白胶卷
- 背包最好预留 `1` 个空位，用于生成已曝光胶卷
- 如果刚更新资源包或贴图但游戏没变化，优先重启游戏
- 反馈问题时，优先附上 `latest.log` 和 `crash-reports` 中的最新文件
