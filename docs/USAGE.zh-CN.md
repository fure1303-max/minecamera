# MineCamera 使用说明

## 版本要求

- Minecraft：`1.21.11`
- Fabric Loader：`0.19.3`
- Java：`21`

`MineCamera` 当前按以上版本组合开发和验证。

## 物品说明

- `相机`：打开 HUD 取景、拍照和回放照片
- `三脚架`：放置后可挂载相机，作为独立机位使用
- `胶卷`：空白胶卷用于拍摄，已曝光胶卷用于回放
- `照片展示框`：当前版本只提供物品与基础放置形态，完整展示功能还未做完

## 基础操作

### 手持相机

- `右键`：打开相机 HUD

### 相机取景模式

- `左键`：拍照
- `右键`：关闭相机 HUD
- `Tab`：切换分辨率预设
- `鼠标滚轮`：调节焦距
- `Shift + 鼠标滚轮`：快速调节焦距
- `R`：切到回放模式

当前焦距范围：

- 最小：`10mm`
- 最大：`600mm`
- 默认打开相机时：`20mm`

### 相机回放模式

- `A`：上一张
- `D`：下一张
- `R`：返回取景模式
- `右键`：关闭相机 HUD

## 三脚架操作

- 手持 `三脚架` `右键`：放置三脚架
- 手持 `相机` 对空三脚架 `Shift + 右键`：安装相机
- 对已安装相机的三脚架 `右键`：打开三脚架相机
- 空手对已安装相机的三脚架 `Shift + 右键`：拆下相机

## 胶卷规则

- 背包里有 `空白胶卷` 时才能拍照
- 每次成功拍照会消耗 `1` 个空白胶卷
- 拍照后会生成 `1` 个已曝光胶卷
- 已曝光胶卷可在相机 `回放模式` 中查看

## 照片展示框

- 手持 `照片展示框` `右键`：放置展示框
- 当前版本暂未完成：
  - 将照片装入展示框
  - 展示框直接显示照片
  - 多个展示框自动拼组大图展示

## 合成配方

### 相机

```text
玻璃片    铁锭    玻璃片
铁锭      钻石    铁锭
空        铁锭    空
```

### 三脚架

```text
空        铁锭    空
空        木棍    空
木棍      空      木棍
```

### 胶卷

无序合成：

```text
纸 + 纸 + 黑色染料 = 胶卷 x2
```

### 照片展示框

```text
木棍      纸      木棍
纸        空      纸
木棍      纸      木棍
```

## 照片保存位置

通用路径：

```text
<游戏目录>\versions\<当前版本>\minecamera\captures\
```

文件名格式：

```text
<mediaId>.png
```

## 常见问题

### 为什么能放展示框，但不能把照片展示出来？

因为这一部分当前还没实现，现阶段只有物品和基础放置形态。

### 为什么我切换到别的 Fabric 版本会出问题？

因为这个模组当前是按 `Fabric Loader 0.19.3` 验证的，不承诺对其他 Loader 版本兼容。

### 如果反馈问题，应该给什么文件？

优先提供：

- `latest.log`
- 对应时间点的崩溃报告
- 是否装了 `Sodium / Iris / 光影包`

日志路径通常是：

```text
<游戏目录>\versions\<当前版本>\logs\latest.log
```

崩溃报告通常在：

```text
<游戏目录>\versions\<当前版本>\crash-reports\
```

有些启动器也可能把错误导出到：

```text
C:\Users\<你的用户名>\AppData\Local\Temp\
```
